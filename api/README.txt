----------- Prérequis  

Avant de commencer, il faut s'assurer d'avoir les éléments suivants installés :

- [Node.js] : https://nodejs.org/ (incluant npm)
- [MongoDB] : https://www.mongodb.com/try/download/community



----------- Installation

Installation des dépendances :

   - Ouvrez un terminal et naviguez jusqu'au répertoire du projet :
     cd chemin/vers/ le projet
  
   - Installez les dépendances du projet :
     npm install
 
     Cela installera les dépendances suivantes :
     - Express: 4.18.2
     - Mongoose: 7.4.2
     - Multer: 1.4.5-lts.1


----------- Utilisation  

1. Démarrage de MongoDB:
   - Démarrez votre serveur MongoDB.

2. Démarrage du Serveur Express:
	- Démarrez le serveur Express :
     npm start


----------- Tests  

Pour exécuter les tests, utilisez la commande suivante dans un terminal :
sh test.sh
